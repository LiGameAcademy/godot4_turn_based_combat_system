### 1.3 角色配置项说明

| 配置项 | 类型 | 说明 | 可选值/示例 |
|--------|------|------|------------|
| `character_name` | StringName | 角色名称 | "勇者", "哥布林" |
| `description` | String | 角色描述 | "一位勇敢的冒险者..." |
| `element` | int | 元素类型 | 0=无, 1=火, 2=水, 3=土, 4=光 |
| `attribute_set_resource` | SkillAttributeSet | 属性集合 | 包含所有属性定义 |

| `sprite_offset` | Vector2 | 精灵偏移 | 调整显示位置 |
| `ai_behavior` | AIBehavior | AI行为配置 | 仅敌人角色需要 |

### 2.3 技能配置项说明

| 配置项 | 类型 | 说明 | 可选值/示例 |
|--------|------|------|------------|
| `skill_id` | StringName | 技能唯一标识 | `"attack"`, `"fireball"` |
| `skill_name` | String | 技能显示名称 | "普通攻击", "火球术" |
| `description` | String | 技能描述 | "向敌人投掷一个火球..." |
| `skill_type` | SkillType | 技能类型 | ACTIVE, PASSIVE |
| `is_melee` | bool | 是否近战 | true, false |
| `mp_cost` | int | 魔法消耗 | 0, 5, 8, 10... |
| `target_type` | TargetType | 目标类型 | 见下方目标类型表 |
| `target_count` | int | 目标数量 | 1-10 |
| `can_target_dead` | bool | 可对死亡目标施放 | true, false |
| `effects` | Array[SkillEffect] | 效果列表 | 技能的所有效果 |
| `action_categories` | Array[String] | 行动类别 | 见下方行动类别表 |
| `icon` | Texture2D | 技能图标 | 图标资源 |
| `cast_animation` | StringName | 施法动画 | `"attack"`, `"projectile_cast"` |
| `pre_cast_delay` | float | 释放前延迟（秒） | 0.0, 0.2, 0.5... |
| `post_cast_delay` | float | 释放后延迟（秒） | 0.0, 0.3... |

### 2.6 技能效果类型配置

#### 伤害效果 (DamageEffect)

| `defense_power_scale` | float | 防御力加成系数 | 0.0, 1.2... |
| `apply_damage_random` | bool | 是否应用伤害随机 | true, false |
| `damage_random_range` | float | 伤害随机范围 | 0.1 (±10%) |
| `element` | int | 元素属性 | 0=无, 1=火, 2=水, 3=土, 4=光 |
| `pre_cast_delay` | float | 释放前延迟 | 0.0, 0.2... |
| `post_cast_delay` | float | 释放后延迟 | 0.0, 0.3... |

#### 治疗效果 (HealEffect)

| 配置项 | 类型 | 说明 | 示例值 |
|--------|------|------|--------|
| `heal_amount` | int | 基础治疗值 | 10, 20, 50... |
| `heal_power_scale` | float | 魔法攻击力加成系数 | 0.0, 0.5, 1.0... |

#### 施加状态效果 (ApplyStatusEffect)

| 配置项 | 类型 | 说明 | 示例值 |

| `status_stacks_to_apply` | int | 堆叠层数 | 1, 2, 3... |

#### 多段攻击效果 (MultiStrikeEffect)

| 配置项 | 类型 | 说明 | 示例值 |
|--------|------|------|--------|

- 触发效果: CounterAttackEffect
  - 反击技能: `attack` (普通攻击)
  - 释放前延迟: 0.0
  - 触发条件: 概率条件（具体概率未明确）
